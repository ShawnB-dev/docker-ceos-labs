# Lab 08 – Rust によるテレメトリ収集

## 1. 概要
このラボでは、Rust を使用してテレメトリ収集パイプラインを実装しました。ラボ 07 の自動化ツールを、インターフェースのカウンタや BGP プロトコルの状態を定期的に取得する永続的なコレクターに進化させました。

## 2. トポロジー
- leaf-1 (10.99.0.21)
- leaf-2 (10.99.0.22)
- rust-telemetry (10.99.0.50)

## 3. 実装の詳細
- **定期的なポーリング**: `tokio::time::sleep` を使用して、30秒ごとにネットワークをポーリングします。
- **JSON-RPC パース**: `inOctets`、`outOctets`、および BGP の `peerState` を抽出するために、複雑なネストされた JSON 構造をパースしました。
- **可視化**: リアルタイムの統計情報がコンソールおよびコンテナログに出力されます。

## 4. クイックスタート

cd 08-telemetry
