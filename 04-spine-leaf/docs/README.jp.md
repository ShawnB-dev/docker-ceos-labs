# ラボ 04 – スパイン・リーフ ファブリック (L3 アンダーレイ)

## 1. 概要
このラボでは、Docker 上で動作する Arista cEOS コンテナを使用して、最小限かつ現実的な 2 スパイン × 4 リーフの L3 アンダーレイファブリックを構築しました。

実装した主な機能：
- 純粋な L3 ルーテッドアンダーレイ
- ファブリック全体での OSPF エリア 0 の適用
- 将来のオーバーレイ（ラボ 06 の EVPN）用のループバックインターフェース
- 両方のスパインにわたる ECMP（等コストマルチパス）
- 隔離された p2p ネットワークを使用した `docker-compose` の配線

## ルーティング
すべてのリンクで OSPF エリア 0 を実行しています。
- ループバックはパッシブ設定
- ECMP は自動的に有効化（spine1 と spine2 を介した 2 つの等コストパス）
- 管理用 IP:
  - spine1: 10.99.0.11
  - spine2: 10.99.0.12
  - leaf1: 10.99.0.21
  - leaf2: 10.99.0.22
  - leaf3: 10.99.0.23
  - leaf4: 10.99.0.24

## 起動手順
1. ファブリックの開始
```bash

```
2. コンテナの確認
```bash
docker ps --format "table {{.Names}}\t{{.Status}}"
```
